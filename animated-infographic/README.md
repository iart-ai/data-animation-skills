# Animated Infographic

Build a designed infographic — icons, a few key numbers, short text, and simple shapes arranged with real visual hierarchy — then animate it in a staggered cascade so each element earns its moment. For analysts, PMs, and data storytellers who want a composed "by the numbers" scene where the static layout reads on its own and the motion just guides the eye.

## When it activates

- "Make an animated infographic" or "create an infographic video."
- "Do a sequenced reveal of icons and stats" or "animate icons with key numbers and connectors."
- "Animate an explainer infographic" or "lay out a designed infographic that animates in sequence."
- "By the numbers" / "X in 3 steps" / stat-card grid scenes.

## Example prompts

- "Make an animated 'how it works' infographic with 3 steps, icon pops, and connectors."
- "Build a 'by the numbers' scene with 4 stat cards that count up one at a time."
- "Animate an icon-plus-stat cascade for our quarterly highlights."

## What's inside

- `SKILL.md` — design-the-static-frame-first rule, visual hierarchy → reveal order, the stagger cascade, icon pops, settling count-up numbers, draw-on connectors, and section pacing budgets.
- `references/sequenced-infographic.md` — a complete runnable Remotion scene: a 3-step "how it works" infographic with staggered icon pops, eased count-up stats, draw-on connectors, a master timing map, the template × data prop pattern, and a dependency-free inline-SVG variant.
